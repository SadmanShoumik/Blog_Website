# Blog Website
Project to create a basic web blog.

## Features
1. User Registration and Login
2. Reading Blogs by Other Authors
3. Searching Blogs by Keyword
5. Creating a New Blog Post
6. Editing a Post
7. Deleting a Post
8. Updating User Information
